# A Food app with `React-Native` + `NativeBase` 🍟

### Technologies ✨

- `React-Native`, works on both `ios` and `android`
- `Expo`
- `NativeBase`
- `@react-navigation/bottom-tabs` and `@react-navigation/native-stack`
